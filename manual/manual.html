Назначение системы

Данное приложение представляет собой систему поиска оптимального маршрута по карте города.

Оно позволяет пользователю сконструировать топологию улично-дорожной сети, сохранить созданную карту или загрузить уже существующую. Система производит поиск оптимального маршрута между двумя точками карты, выбранными пользователем, с заданными параметрами автомобиля и критерием поиска.

Условия работы системы

Для корректного функционирования приложения необходимо:
  • тип ЭВМ: x86-64 совместимый;
  • объем ОЗУ – не менее 1,4 Гб
  • объем свободного дискового пространства – не менее 18,4 Гб
  • процессор – Intel Pentium не менее 1,5 ГГц
  • Java 8
  • операционная система: Windows 10 (8u51 или более поздняя), Windows 8.x (настольная версия), Windows 7 с пакетом обновления 1 (SP1), Mac OS X 10.8.3+ или 10.9+, Oracle Linux 5.5+, Ubuntu Linux 12+
  • дисплей с разрешением не менее 1024 × 768 пикселей
  • клавиатура или иное устройство ввода
  • мышь или иное манипулирующее устройство.

Установка системы

Для запуска системы необходимо установить JDK. Руководство по установке приведено на сайте: http://java-course.ru/begin/install-jdk/.

Работа с системой

Конструирование карты

Для того, чтобы начать конструировать карту, нужно открыть вкладку «Редактор».

Разместить перекресток

Для размещения перекрестка необходимо нажать кнопку «Перекресток» в верхней части боковой панели (рисунок А.1), затем щелкнуть левой кнопкой мыши на область карты, расположенной в центре рабочей области, где необходимо разместить перекресток.

Разместить дорогу

Для размещения дороги необходимо нажать кнопку «Дорога» в верхней части боковой панели (рисунок А.1), затем последовательно щелкнуть левой кнопкой мыши на два перекрестка, которые необходимо соединить дорогой.

Задать параметры перекрестка

Для того, чтобы задать параметры перекрестка, необходимо щелкнуть левой кнопкой мыши на размещенный на карте перекресток, система отобразит форму для настройки его параметров (рисунок А.2), на которой пользователь может установить флажок для поля «Светофор», тем самым поставив светофор на перекресток, и затем задать длительность красной и зеленой фаз. Для сохранения настройки параметров пользователю необходимо нажать на кнопку «Сохранить».

Удалить перекресток

Для удаления перекрестка с карты необходимо щелкнуть левой кнопкой мыши на размещенный на карте перекресток, система отобразит форму для настройки его параметров (рисунок А.2), на которой нужно нажать кнопку «Удалить». После этого система запросит подтверждение действия (см. рисунок А.3). Система удалит перекресток вместе с прилегающими к нему дорогами.

Задать параметры дороги

Для того, чтобы задать параметры дороги, необходимо щелкнуть левой кнопкой мыши на размещенную на карте дорогу, система отобразит форму для настройки ее параметров (рисунок А.2), на которой нужно нажать кнопку «Удалить». После этого система запросит подтверждение действия (рисунок А.4), на которой пользователь может выбрать направление движения, название улицы, ограничение скоростного режима и тип покрытия дороги. Для сохранения настройки параметров пользователю необходимо нажать на кнопку «Сохранить».

Удалить дорогу

Для удаления дороги с карты необходимо щелкнуть левой кнопкой мыши на размещенную на карте дорогу, система отобразит форму для настройки ее параметров (рисунок А.4), на которой нужно нажать кнопку «Удалить». После этого система запросит подтверждение действия.

Переместить перекресток

Для перемещения перекрестка необходимо щелкнуть левой кнопкой мыши на размещенный на карте перекресток и, не отпуская кнопки мыши, переместить перекресток на новое место.

Изменить масштаб карты

Изменить масштаб карты можно нажимая на иконки «+» и «-», либо перемещая ползунок, расположенный между иконками в верхней левой части рабочей области (рисунок А.1).

Сохранение карты

Для того, чтобы сохранить карту необходимо щелкнуть кнопку «Сохранить в файл» в нижней части боковой панели на вкладке «Редактор» (рисунок А.1). Затем ввести имя файла, путь для сохранения карты и щелкнуть кнопку «Сохранить».

Загрузить существующую карту для редактирования

Для того, чтобы загрузить существующую карту для редактирования необходимо щелкнуть кнопку «Загрузить из файла» в нижней части боковой панели на вкладке «Редактор» (рисунок А.1). Затем выбрать файл с нужной картой и щелкнуть кнопку «Открыть».

Построение маршрута

Для того, чтобы начать построение маршрута, нужно открыть вкладку «Поиск маршрута» и загрузить файл карты, нажав на кнопку «Открыть файл карты» в верхней части боковой панели и выбрав нужный файл (рисунок А.5). Далее необходимо выбрать начальную и конечную точки маршрута и нажать кнопку «Найти» в нижней части боковой панели. Если маршрут будет найден, то он отобразится на карте, иначе система отобразит сообщение о том, что маршрут не может быть построен. При построении маршрута в нижней части рабочего окна отобразится значение критерия поиска для найденного маршрута. Система может отобразить значения критерия для каждого участка дороги, для этого необходимо установить флажок поля «Отобразить значения на карте».

Задать точки маршрута

Для того, чтобы выбрать маршрут необходимо выбрать начальную и конечную точки. Для выбора начальной точки маршрута необходимо нажать левой кнопкой мыши на перекресток и выбрать пункт «Отсюда» из выпадающего меню. Для выбора конечной точки маршрута необходимо нажать левой кнопкой мыши на перекресток и выбрать пункт «Сюда» из выпадающего меню (рисунок А.5).

Задать параметры маршрута

Для задания параметров маршрута необходимо выбрать соотвествующее значение критерия, водителя и автомобиля с помощью выпадающего меню на боковой панели (рисунок А.5).

Изменить масштаб карты

Изменить масштаб карты можно нажимая на иконки «+» и «-», либо перемещая ползунок, расположенный между иконками в верхней левой части рабочей области (рисунок А.5).

Ведение БД

Для работы с БД необходимо выбать вкладку «Работа с БД» (рисунок А.6). В верхней части данной вкладки расположена панель, вкладки которой соответствуют различным параметрам, использующимися в системе. При выборе вкладки отображается таблица с соответствующими данными, которые хранятся в системе.

Добавление записи

Для добавления новой записи необходимо заполнить данными все поля в нижней части рабочего окна и нажать кнопку «Добавить» (рисунок А.6). Новая запись отобразится в таблице.

Изменение записи

Для редактирования записи необходимо выбрать ее двойным щелчком левой кнопкой мыши и ввести новые значения данных (рисунок А.6).

Удаление записи

Для удаления записи необходимо ее выбрать, нажав на нее левой кнопкой мыши, затем нажать кнопку «Удалить» (рисунок А.6).
